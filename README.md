# Dashboard

Es una aplicación para gestionar datos energéticos en tres aplicaciones diferentes.

# Estorninos

Es la aplicación utilizada en el proyecto Estorninos de Som Energía.
El objetivo es visualizar las variables que se consideran oportunas para determinar los perfiles de consumo mas adecuados con algunos dias de anticipación.

# Tomorrow

Es la aplicación utilizada de forma individual para decidir sobre los mejores momentos para hacer consumo de electricidad basandose en los precios de la tarifa indexada de la cooperativa y la producción fotovoltaica local. A efectos de deternminar la potencialidad de autoconssmo se muestra la curva típica de potencia disponible del dia de hoy según promedios de PVGIS y el pronostico meteorológico con especial atención a la nubosidad.

# Yesterday

Es la aplicación uililizada para la estadística pasada
